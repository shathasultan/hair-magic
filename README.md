# سحر الصبغة - Hair Magic

موقع احترافي عن فن اختيار ألوان الصبغات ورعاية الشعر

## 🌟 الميزات

- ✨ تصميم أنيق وحديث
- 📱 تصميم responsive يعمل على جميع الأجهزة (الهاتف، الايباد، الكمبيوتر)
- 🎨 لوحة ألوان راقية (ورديّ، ذهبي، بني)
- 🎯 8 أقسام رئيسية:
  - اختيار اللون المثالي حسب لون البشرة
  - أنواع الصبغات المختلفة
  - الكيمياء والمكونات
  - الأضرار المحتملة والوقاية
  - العناية اللاحقة والصيانة
  - فريق الخبراء
  - وأكثر!

## 📲 الأجهزة المدعومة

- ✅ الهاتف الذكي (320px فما فوق)
- ✅ الجهاز اللوحي iPad (768px - 1024px)
- ✅ الكمبيوتر المحمول (1024px - 1200px)
- ✅ شاشات سطح المكتب الكبيرة (1200px+)

## 🎨 التقنيات المستخدمة

- HTML5
- CSS3 (مع CSS Variables)
- JavaScript (ES6)
- SVG Graphics
- اللغة العربية (RTL)

## 📝 الخطوط المستخدمة

- Playfair Display (للعناوين الرئيسية)
- Cormorant Garamond (للتفاصيل الأنيقة)
- Great Vibes (للشعارات)
- Amiri (للنصوص العربية)
- Scheherazade New (للنصوص العربية الإضافية)
