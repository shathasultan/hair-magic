# ⚡ البدء السريع

## للرفع على GitHub في 5 دقائق:

### 1️⃣ إنشاء Repository على GitHub

```
اذهب إلى https://github.com/new
أدخل الاسم: hair-magic
اختر: Public
اضغط: Create Repository
```

### 2️⃣ نسخ أوامر git (ستجدها على الصفحة):

```bash
# استبدل YOUR_USERNAME باسمك على GitHub
cd /Users/shathasultan/project

git remote add origin https://github.com/YOUR_USERNAME/hair-magic.git
git branch -M main
git push -u origin main
```

### 3️⃣ تفعيل GitHub Pages

```
Settings → Pages → Source: main → Save
```

### 4️⃣ الانتظار 1-3 دقائق ثم الدخول إلى:

```
https://YOUR_USERNAME.github.io/hair-magic/
```
